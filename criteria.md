# Критерии оценки Модуля В (Аудит смарт-контракта)

Обозначения:
- **С** — субъективный критерий (оценивается экспертом по шкале).
- **И** — измеримый (бинарно: выполнено / не выполнено).
- Балл за критерий = его **вес**. Итог — сумма весов по всем зачтённым критериям.

## Субъективные

| Код | Критерий | Балл | Шкала |
|---|---|---|---|
| С | Качество представленных отчетов | 0,50 | 0 — не представлены/не читаемы; 1 — посредственно; 2 — без исчерпывающих комментариев; 3 — исчерпывающие пояснения |
| С | Описан функционал контракта | 0,30 | 0 — не описан; 1 — общие слова; 2 — развёрнуто, но с упущениями; 3 — исчерпывающее описание |
| С | Дополнительные предложения по оптимизации | 0,30 | 0 — нет; 1 — слабые; 2 — заметные; 3 — кардинально оптимизируют |
| С | Предложения по оптимизации не скажутся негативно | 0,30 | 0 — кардинально негативные; 1 — значительно негативно; 2 — незначительно негативно; 3 — негативных нет |
